# Life-Build
Somos o seu site de rotinas, a ferramenta definitiva para organizar, monitorar e otimizar seu dia a dia.  Simples, intuitivo e eficaz. Transforme metas em hábitos e produtividade em estilo de vida.

Para utilizar o site de maneira segura e funcional, utilize os seguintes códigos:

pip install Flask
# Instala o Flask

pip install Flask-SQLAlchemy
# Instala o Flask-SQL para o banco de dados
